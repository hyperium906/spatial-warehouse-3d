Agile Development Roadmap: 3D warehouse pipeline

Track the iterative delivery milestones of this spatial computer vision application built across 5 two-week Agile Sprints.

Sprint 1

Core 3D Viewer & Layout

  • Rendered mock warehouse grid layout and lighting setup.
  • Integrated OrbitControls for panning, zooming, and orbiting.
  • Implemented basic primitive geometry mocks for storage shelves and aisles.
Sprint 2

Object Bounding Box Engine

  • Engineered Python CV simulation engine with AABB (Axis-Aligned Bounding Box) math.
  • Added support for 3D raycasting in Three.js to pick objects and retrieve metrics.
  • Drafted initial database.json structure for static spatial persistence.
Sprint 3

Meshroom CLI Automation

  • Automated photogrammetry sequence in CLI from python backend.
  • Built asynchronous background workers to execute Node pipeline (SfM, Meshing, Texturing).
  • Integrated CLI stderr/stdout logging streaming via REST API.
Sprint 4

Spatial Updating & Collisions

  • Integrated manual coordinate overrides using HTML slider panels.
  • Implemented real-time collision detection calculations on spatial updates.
  • Added safety diagnostics warnings when aisle blockages or object intersections occur.
Sprint 5

Full Inventory Dashboard & UI

  • Polished premium dark-slate design with glassmorphic cards and glowing metrics.
  • Aggregated inventory stats, rack occupancy volumes, and space utilization gauges.
  • Wrote API hooks to sync floor updates persistently, allowing deletion and additions.
